# Antigravity Mobility Device Showcase Site

A premium, high-fidelity landing page showcasing the **Antigravity Mobility Device**—a hybrid mobility solution designed to provide an effortless, low-strain experience through smart torque engineering.

![Project Preview](/public/hero.png)

## 🚀 Overview

This site is a digital twin of the "Antigravity" product concept. It utilizes a modern, "Deep Space" aesthetic to communicate the product's core values: **Lightweight, Powerful, and Seamless.**

### Key Features

- **Interactive Assist Visualizer**: A state-driven simulation component that demonstrates the balance between user effort and motor assistance.
- **Responsive "Glassmorphism" UI**: A premium design system built with custom Vanilla CSS filters and HSL color tokens.
- **Product Strategy Integration**: Incorporates the case study's core pillars, feature prioritization, and engineering specs.
- **Fluid Motion**: Implements high-performance micro-animations using Framer Motion.

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Vanilla CSS (Modern CSS Variables, HSL, Glassmorphism)
- **Icons**: Lucide React
- **Animations**: Framer Motion

## 🚦 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm (v9 or higher)

### Installation

1. Navigate to the project directory:
   ```bash
   cd antigravity-site
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production

To create a production-ready bundle:
```bash
npm run build
```

## 📋 Strategic Context

This landing page was built to support a product management case study for the personal mobility market. The goal was to prove the viability of a "Missing Middle" product segment that bridges the gap between manual wheelchairs and heavy-duty electric scooters.
